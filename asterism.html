星辰光アステリズム

星辰光（アステリズム）は、プラネテスルールにおける特別な能力や力を指します。
これらの力は、星辰光が特定の条件を満たすことで発動し、戦闘や探索、諜報など様々な場面で役立ちます。

星辰光の種類

赤の星辰光

熱情

種類:補助動作 / 前提:無し
形状：無し / 時間:10秒
コスト:MP1点

感情を昂らせます。
赤を1点獲得します。
自身が合計で50点以上のダメージを与えるか回復した時、追加で赤を1点獲得します。
この効果は1度の手番に1回使用できます。

激昂

種類:補助動作 / 前提:赤3点
形状：無し / 時間:10秒
コスト:無し

技の威力を高めます。
与ダメージに+4のボーナスを得ます。
回避力に-2の修正を得ます。

紅蓮

種類:補助動作 / 前提:赤3点
形状：無し / 時間:10秒
コスト:無し

クリティカルが発生した時、1度だけ追加ダメージを8点得ます。

獅子奮迅

種類:補助動作 / 前提:赤5点
形状：無し / 時間:10秒
コスト:無し

技の威力を高めます。
与ダメージに+8の修正を得ます
この効果の間、自身の回避力に-2の修正を得ます。

研ぎ澄ました一撃

種類:補助動作 / 前提:赤5点
形状：無し / 時間:10秒
コスト:無し

クリティカルが発生した時に1度だけ追加ダメージに16点を得ます。

情熱と自由と勝利の決断

種類:補助動作 / 前提:赤7点
形状：無し / 時間:10秒
コスト:無し


自身の与ダメージに+25点の修正を得ます。
この効果の間、自身がダメージを与えるたび7点の確定ダメージを受けます。

戦神の猛りよ、勝利を掴み取れアレスレイジ・オーバードライブ

種類:補助動作 / 前提:赤7点
形状：無し / 時間:10秒
コスト:


威力表を決定する際、その出目に+3する。
この効果の間、自身の回避力に-2の修正を受けます

青の星辰光

瞑想

種類:補助動作 / 前提:無し
形状：無し / 時間:無し
コスト:MP1点

冷静に自己と向き合います。
次の手番の初めに青を1点獲得します。
効果時間中に判定を自動成功か1ゾロによる自動失敗していた場合、追加で青を1点獲得します。
1度の手番に1回使用できます。

洞察

種類:補助動作 / 前提:青3点
形状：無し / 時間:10秒
コスト:無し

冷静に相手に対処します。
任意の能力値ボーナス1つに+2の修正を得ます

最適化

種類:補助動作 / 前提:青3点
形状：無し / 時間:10秒
コスト:無し

このラウンドで消費するあらゆるMPに-1点の修正を与えます。

行雲流水こううんりゅうすい

種類:補助動作 / 前提:青5点
形状：無し / 時間:10秒
コスト:

自身の身体を戦闘に適応した形に変化させます。
命中力、行使判定にそれぞれ1度だけ+2の修正を与えます。

連続魔法

種類:補助動作 / 前提:青5点
形状：無し / 時間:10秒
コスト:無し

このラウンド中に魔法を使用する度に、与ダメージに+4の修正を得る。

叡智と流転と求道の決断

種類:補助動作 / 前提:青7点
形状：無し / 時間:10秒
コスト:無し

自身の身体を一時的に最適化します
判定を任意の能力値Bで行う。
全ての能力値に+4のボーナスを得る。

永劫の神よ、超光で刹那に無限をタキオンループ・アイオーン

種類:補助動作 / 前提:青7点
形状：無し / 時間:一瞬
コスト:無し

一時的に限界を超え加速します
この手番の次にもう一度手番を行います

緑の星辰光

種類:補助動作 / 前提:無し
形状：無し / 時間:無し

1点の緑を獲得します。

この効果は1度の手番に1回使用できます。

形状：1体 / 時間:一瞬
コスト:無し

献身の絆

形状：1体 / 時間:無し
コスト:無し

対象が受けている「精神効果属性」または「毒・病気属性」の効果を1つ解除する。
HPを10点回復する。
この能力は1度の手番に1度だけ使用できます。

琴瑟調和きんしつちょうわ

コスト:無し

種類:補助動作 / 前提:緑7点
形状：無し / 時間:10秒
コスト:無し

任意の色1つを選び、その色を緑と同じ点数にする。

この能力は1度の手番につき1度使用できる

形状：2エリア/10体 / 時間:10秒
コスト:無し

この効果が発動する度、術者はHPとMPを7点回復します。
